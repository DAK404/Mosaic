Features!

This program/toolset has many built-in features which include:

  • NEW! Supports Windows® OS, Mac® OS and Linux OS!
  • Mosaic Kernel: A successor to the Elixir Kernel, which improves security, performance and reliability!
  • Anima Database API tools, created afresh from start!
    NOTE: Fractal File Saving System is deprecated now. It is no more supported.
  • "Download API" to download, encrypt, decrypt files! Secure your important files!
  • Upgraded Updater program, to provide a seamless update process! the user now needs to sign out and sign in to apply changes!
  • Text Editor: Want to write your own files? Here you go! Text Editor is your notepad in Mosaic to help in securing confidential info!
  • Censoring system in chat is stable as ever!
  • Updates are now streamlined to a unified build! Which means any platform can run the program!
  • Added BuildScript, Run_Program (for Windows® OS), Run_Program_ WSL (for Windows® OS with WSL installed)
  • Many many more features in work! Stay Tuned to know more!
