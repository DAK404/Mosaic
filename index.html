Features!

This program/toolset has many built-in features which include:

  • Unified Build          : Run the same build on any platform/OS now! (Tested on Windows®, MacOS® and Linux)
  • Mosaic Kernel       : A simple, secure, modular and efficient framework. Now has APIs to support 3rd party programs!
  • Anima Database   : SQLite database used to manage the database containing credentials, under the Fractal Credential and Accounts Database.
  • Program Updater  : Helps to update the program to the latest build from the given sources. Seamless OTA update, applied right after a program restart.
  • Built-in Tools          : Read/write/modify/download/manage files.
  • Built in scripts        : Scripts which helps to Compile, debug and document the program components with scripts!
