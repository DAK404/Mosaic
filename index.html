Mosaic: A Nion Project

|Mosaic: A new project to create a better framework which Elixir Kernel offers! | This Framework is being modularized more, making the program smaller and widening user's preferences! |

Mosaic is a project which aims at a platform which is flexible, robust and secure!
This program has built in powerful tools to work with and to work on, making it a platform to host, deploy and maintain with ease!

Features!

This program/toolset has many built-in features which include:

  • Works on a new Mosaic Kernel, a successor of Elixir Kernel. Optimized memory usage and program efficiency!
  • Anima Database API tools, created afresh from start!
    NOTE: Fractal File Saving System is deprecated now. It is no more supported.
  • Download API to download, encrypt, decrypt files! Secure your important files!
  • Upgraded Updater program, to provide a seamless update process! the user now needs to sign out and sign in to apply changes!
  • Text Editor: Want to write your own files? Here you go! Text Editor is your notepad in Mosaic to help in securing confidential info!
  • Censoring system is stable as ever!
  • Updates are now streamlined to a unified build! Which means any platform can run the program!
  • Added BuildScripts, ResetScripts, RunScripts and many other scripts to help in debugging and automating tasks!
  • Many many more features in work! Stay Tuned to know more!
